
Python: RSS reader
 
Приложение должно позволять пользователям хранить и просматривать RSS ленты. Для пользованием приложением необходима авторизация пользователей. Пользователь не видит данных других пользователей.
 
Пользователь может вносить новые ленты, просматривать, удалять и редактировать имеющиеся, видеть список собственных лент. Вносить RSS ленту необходимо по URL.

Просмотр ленты должен выводить список с элементами ленты (название, тело, линки на локальные файлы). 

Приложение должно скачивать локально RSS feed XML и все документы, на которые ссылается этот feed (URL из link элементов для каждого item). Feed XML надо преобразовывать, чтобы ссылки указывали на локальные копии документов.
 
Для упрощения, преобразование собственно документов (например, правка relative hyperlinks) с целью их корректного отображения НЕ требуется.
 
Приложение должно учитывать следующие моменты:
 
необходимо учитывать, что несколько feeds могут ссылаться на один и тот же документ (см. примерный список feed URLs, одна из последних статей вполне может быть в top list) -- не надо скачивать один и тот же документ повторно
приложение должно периодически обновлять ленты
исходные документы МОГУТ изменяться, но приложение должно корректно обновлять локальный документ при изменении удаленного
через некоторое время ссылка на документ исчезает из feed XML, соответственно, отпадает необходимость хранить копию документа локально, не надо создавать локальную копию всего Internet,неиспользуемые документы должны удалятся
необходима корректная обработка сетевых ошибок
все списки должны иметь пагинацию по 10 элементов на страницу
