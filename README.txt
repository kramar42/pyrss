
Python: RSS reader
 
Сейчас есть возможность регистрировать новых пользователей. 
Проверки на существование пользователя с таким же именем пока что нету.

В главном меню пользователя (/feeds) доступен список из всех RSS-записей пользователя.
Реализована возможность добавлять новые записи по URL-адрессу.
Записи храняться в виде файла в таблице.
Записи между разными пользователями не пересекаються.

Записи закачиваются в локальную директорию ./pyrss/feeds

Запуск django-project-a:
	cd pyrss; ./manage.py startserver

Страница логина:
    localhost:8000/login/

Тестовый пользователь:
    Test Test
 
Примеры RSS лент для тестирования:
    http://rss.cnn.com/rss/cnn_topstories.rss
    http://rss.cnn.com/rss/cnn_latest.rss
